## How to build

Clone the repo.

Don't forget `$> npm init`.

Override the [settings](./settings) to your `.clasp.json`.

Push the snippet `sheets/export-import-to-csv` to the cloud:

```sh
$>  S=./snippets/sheets/export-import-to-csv && npm run dsw $S
```
